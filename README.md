# 人群计数系统 前台代码

## 项目介绍

人群计数系统前台代码，基于streamlit开发。

## 项目运行

``` bash
# 安装依赖
pip install -r requirements.txt

# 运行
streamlit run ./frontend/__init__.py
```

## 项目结构

``` bash
.
├── frontend
│   └── __init__.py # 启动文件
├── poetry.lock
├── pyproject.toml
├── README.md # 项目说明
├── requirements.txt
└── tests
    └── __init__.py

2 directories, 6 files
```
