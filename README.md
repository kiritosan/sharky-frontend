# 人群计数系统 前台代码

## 项目介绍

人群计数系统前台代码，基于streamlit开发。

## 项目运行

``` bash
# 安装依赖
pip install -r requirements.txt

# 运行
streamlit run ./frontend/__init__.py
```

## 项目结构
